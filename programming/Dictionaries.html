#6 - Dictionary

Needing to edit dictionaries is not very common, but sometimes it is useful. Write a function named get_older that consumes a dictionary (representing a person) and increases its "Age" key's value by 1. The function should return the new age. Call the function on the two given person dictionaries to advance their ages, and print their returned values.

Note: This function modifies the dictionary that it consumes. This is unusual and different from what we've done in the past, creating new dictionaries or lists. We sometimes want to modify dictionaries, but keep in mind that this makes it harder to reason about the program over all.


# Starting Code:
person1 = {"Name": "Charles Babbage", "Age": 17}
person2 = {"Name": "Ada Lovelace", "Age": 32}
