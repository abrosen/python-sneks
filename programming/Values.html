#1 - Integers

We can use Python to represent information. Use the NUMBER block (in the "Values" tab) to represent your current age as an integer. Then, print that number.

#2 - Floats

We can use Python to represent information. Type a float number to represent your ideal GPA (e.g., 4.00). Then, print that number. Do not use the BlockPy block - actually write out the number.

#3 - Strings

We can use Python to represent information. Use the STRING block (in the "Values" tab) to represent and print your name. It's okay if you want to use a nickname instead!

#4 - Booleans

We can use Python to represent information. Use the BOOLEAN block (in the "Values" tab) to represent whether you think it will rain tomorrow.
