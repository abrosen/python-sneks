The following definitions and examples are meant to be human-friendly explanations. They are not meant to be formal definitions.

Function

Definition: Reusable chunk of code that contains instructions.

In a sentence: "The function main() runs my program"

Example:

def main():

Parameter

Definition: Placeholder (variable name) for value passed into a function.

In a sentence: "The function main only takes one parameter."

              *note: “takes” is the same as “consumes”

Example:

def main(parameter_1):

Argument

Definition: Value being passed into a function.

In a sentence: "I’m going to call the function main with the argument 13."

Example:

main(13):

Input

Definition: Information taken from the user.

In a sentence: "To unlock my computer, I input my password."

Example:

#*in the console*

Enter your name: BUDDY

Output

Definition: Information produced to a user.

       *note: output can also be what a program printed

In a sentence: "I ran the program and it output text to the console."

Example:

#*code*

print("Hello", name)

#*text displayed in console after running program*

Hello BUDDY

Return

Definition: Value produced by a function.

In a sentence: "The whats_my_name() function returned the string value BUDDY."

Example:

def whats_my_name():
    return "BUDDY"
print(whats_my_name())

#*the console will read*:

BUDDY

Call

Definition: To use a function by putting parentheses after its name.

In a sentence: "I called the print function right here in my code"

Example:

main()
